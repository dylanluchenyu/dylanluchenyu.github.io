Proof Preview is a local HTML viewer for reading and iterating on mathematical proofs. It bundles KaTeX for offline rendering, so proof notes can be previewed without depending on an online service.

The project includes a `proof-preview` command-line helper. Codex can write the latest proof into `current-proof.js`, and the viewer refreshes to show the current proof in a full or compact layout.
